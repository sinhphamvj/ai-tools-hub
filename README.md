# AI Tools Hub

Affiliate content site for AI tools. Auto-generated by Hermes Agent daily.

## Structure
- `_posts/` — articles (YYYY-MM-DD-title.md)
- `_layouts/` — Jekyll templates
- `assets/css/` — styling
- `tools.html`, `comparisons.html`, `posts.html` — index pages

## How content is added

The daily cron `daily-affiliate-flywheel` (job 902bd461d5c8 area) generates new posts and pushes via `git`. No manual editing needed for the daily flow.

## Local preview
```bash
cd ~/projects/ai-tools-hub
bundle exec jekyll serve
# → http://localhost:4000/ai-tools-hub
```

Or just open `https://sinhphamvj.github.io/ai-tools-hub/` after push.
